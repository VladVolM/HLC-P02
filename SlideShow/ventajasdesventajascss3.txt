                <h1>Ventajas de CSS3</h1>
                   <p>
                        <ul>
                            <li>Se obtiene un mayor control de la presentaci&oacute;n del sitio al poder tener todo el c&oacute;digo CSS reunido en uno, lo que facilita su modificación.</li>
                            <li>Al poder elegir el archivo CSS que deseamos mostrar, puede aumentar la accesibilidad ya que podemos asignarle un c&oacute;digo CSS concreto a personas con deficiencias visuales, por ejemplo. Esto lo detecta el navegador web.
                                Conseguimos hacer mucho m&aacute;s legible el c&oacute;digo HTML al tener el c&oacute;digo CSS aparte (Siempre que no usemos estilos en l&iacute;nea, claro est&aacute;).</li>
                            <li>Pueden mostrarse distintas hojas de estilo según el dispositivo que estemos utilizando (versi&oacute;n impresa, versi&oacute;n móvil, leída por un sintetizador de voz…) o dejar que el usuario elija</li>
                            <li>Gracias a la t&eacute;cnica CSS Sprites podemos aligerar la carga de nuestro sitio al juntar todas las im&aacute;genes en una. </li>
                            <li>Las novedades de CSS3 nos permiten ahorrarnos tiempo y trabajo al poder seguir varias t&eacute;cnicas (bordes redondeados, sombra en el texto, sombra en las cajas, etc.) sin necesidad de usar un editor gr&aacute;fico.</li>
                        </ul>
