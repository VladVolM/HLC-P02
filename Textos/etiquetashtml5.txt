			<h1>Nuevas Etiquetas HTML5</h1>
                <p>El lenguaje HTML funciona a trav&eacute;s de marcas de sentido llamadas etiquetas. Las etiquetas son la herramienta fundamental para que los navegadores puedan interpretar el c&oacute;digo y permitirnos ver im&aacute;genes, texto, p&aacute;rrafo, y estructuras. Los navegadores vendr&iacute;an a ser como “traductores” de las etiquetas, y con HTML5, se agregan nuevas etiquetas para utilizar que nos ahorran el uso de otros productos que se usaban para complementar y hacer cosas que con el simple HTML no se pod&iacute;an hacer. HTML5 fue creado para hacer que el proceso de escribir el c&oacute;digo sea m&aacute;s simple y m&aacute;s l&oacute;gico, por decirlo de una forma. La sintaxis de HTML5 se destaca, como dijimos, en el &aacute;mbito multimedia, pero son bastantes las etiquetas introducidas para generar una mejor&iacute;a.</p>
            </article>
            <article>
                <p>La idea detr&aacute;s de HTML5 es que podamos visualizar el contenido multimedia variado que podemos encontrar en internet a&uacute;n cuando nos encontramos en dispositivos de gama baja que no podrían soportarlo cuando tienen que instalar infinidad de plug-ins. No solamente contamos con etiquetas especiales como audio, video y canvas, sino tambi&eacute;n integraci&oacute;n con contenidos de gr&aacute;ficos en vectores (que anteriormente se conoc&iacute;a como la etiqueta object. Con estas etiquetas, los usuarios pueden consumir videos y canciones, por ejemplo, sin necesidad de instalar nada de forma adicional.</p>
            </article>
            <article>
                <p>Las m&aacute;s importantes de las nuevas etiquetas creadas son:</p>
                <ul>
                    <li>article: esta etiqueta sirve para definir un art&iacute;culo, un comentario de usuario o una publicaci&oacute;n independiente dentro del sitio.</li>
                    <li>header, footer: estas etiquetas individuales ahorran tener que insertar IDs para cada uno, como se sol&iacute;a hacer anteriormente. Además, se pueden insertar headers y footers para cada secci&oacute;n, en lugar de tener que hacerlo &uacute;nicamente en general.</li>
                    <li>nav: la negaci&oacute;n puede ser insertada directamente en el markup, entre estas etiquetas, que nos permitir&aacute;n hacer que nuestras listas oficien de navegaci&oacute;n.</li>
                    <li>section: con esta etiqueta, una de las m&aacute;s importantes de las novedades, se puede definir todo tipo de secciones dentro de un documento. Por ponerlo de forma sencilla, funciona de una forma similar a la etiqueta div que nos separa tambi&eacute;n diferentes secciones.</li>
                    <li>audio y video: estas son las dos más importantes etiquetas de HTML5, dado que nos permiten acceder de forma más simple a contenido multimedia que puede ser reproducido por casi todo tipo de dispositivos; marcan el tipo de contenido que estará en su interior.</li>
                    <li>embed: con esta etiqueta se puede marcar la presencia de un contenido interactivo o aplicación externa.</li>
                    <li>canvas: finalmente, esta etiqueta nos permite introducir un “lienzo” dentro de un documento, para poder dibujar gráficos por vectores; será necesario el uso de JavaScript.</li>
                </ul>
