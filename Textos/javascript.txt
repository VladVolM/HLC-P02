                <h1>Definici&oacute;n</h1>
                <p>
                    <p>JavaScript (abreviado com&uacute;nmente JS) es un lenguaje de programaci&oacute;n interpretado, dialecto del est&aacute;ndar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, d&eacute;bilmente tipado y dinámico.</p>
                    <p>Se diseñ&oacute; con una sintaxis similar a C, aunque adopta nombres y convenciones del lenguaje de programaci&oacute;n Java. Sin embargo, Java y JavaScript tienen sem&aacute;nticas y prop&oacute;sitos diferentes.</p>
                    <p>Todos los navegadores modernos interpretan el c&oacute;digo JavaScript integrado en las p&aacute;ginas web. Para interactuar con una p&aacute;gina web se provee al lenguaje JavaScript de una implementaci&oacute;n del Document Object Model (DOM).</p>
                    <p>Tradicionalmente se venía utilizando en p&aacute;ginas web HTML para realizar operaciones y &uacute;nicamente en el marco de la aplicaci&oacute;n cliente, sin acceso a funciones del servidor. Actualmente es ampliamente utilizado para enviar y recibir informaci&oacute;n del servidor junto con ayuda de otras tecnolog&iacute;as como AJAX. JavaScript se interpreta en el agente de usuario al mismo tiempo que las sentencias van descargándose junto con el c&oacute;digo HTML. </p>
                    <p>Desde el lanzamiento en junio de 1997 del est&aacute;cndar ECMAScript, han existido las versiones 2, 3 y 5, que es la m&aacute;s usada actualmente (la 4 se abandon&oacute;). En junio de 2015 se cerr&oacute; y public&acute; la versi&oacute;n ECMAScript 6.7​ </p>
